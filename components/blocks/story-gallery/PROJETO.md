# Bloco de Galeria de Histórias

## Descrição Formal
Componente React para exibição e envio de histórias multimídia, com múltiplas visões, estado local, alimentação por URL JSON, integração com barramento global, mixins de templates e extensibilidade via plug-ins.

### Características
- Herda de `AdvancedBlockBase`
- Estado local e remoto
- Suporte a múltiplas visões (galeria, envio, curadoria)
- Configuração visual via TinaCMS
- Integração com barramento global (window)
- Mixins plugáveis para tipos de mídia e estilos
- Custom code seguro para adaptação de interface
- Alimentação dinâmica por URL JSON
- Suporte a curadoria e moderação
